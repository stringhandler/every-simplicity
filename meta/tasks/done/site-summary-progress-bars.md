# Site: Progress bars on summary pages

On the Jets, Built-ins, Types, and Macros tab tables, add a progress bar to each row that visually shows how frequently that item is used relative to the most-used item in the same table.

This turns the table into a bar chart — each row's bar width is proportional to its script count (max = 100%).

## Acceptance criteria
- Progress bar rendered inline in each row (e.g. behind or below the row label)
- Width scaled relative to the highest count in the table
- Consistent styling with the dark theme
- Works across all four summary tabs (Jets, Built-ins, Types, Macros)
